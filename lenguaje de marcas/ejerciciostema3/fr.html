Guía de Introducción a HTML

El HyperText Markup Language (abreviado como HTML ) es el estándar que define la estructura básica de una página web. Fue creado en los años 90 por Tim Berners-Lee. De acuerdo con W3C, HTML sigue siendo el pilar fundamental del desarrollo web moderno.

Una etiqueta en HTML se marca con un símbolo de mayor > y menor <. Por ejemplo, para hacer un texto en negrita se usa la etiqueta b.

Para escribir código HTML en tu editor de texto, presionaCtrl + S para guardar tu archivo y asegúrate de que la extensión sea html

En este curso, aprenderemos términos clave como elemento semántico y su importancia en el SEO. Además, discutiremos algunas "mejores prácticas" del desarrollo web. Como dijo Tim Berners-Lee: "La Web es más una creación social que técnica."

Anteriormente, en las primeras versiones de la web, algunas etiquetas como marquee y fueron eliminadas. En su lugar, se añadieron nuevas funcionalidades como la etiqueta video.

Conclusión

Entender el uso correcto de las etiquetas HTML no solo mejora la accesibilidad, sino que también proporciona una mejor experiencia de usuario.
